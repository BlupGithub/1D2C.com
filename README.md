# 1D2C.com

Visit the project site at [1D2C.com](https://1d2c.com/).

1D2C is a white-labeled 360° e-commerce platform with apps and tools for buyers, sellers, delivery teams, and admins. It includes features for orders, live tracking, payments, invoicing, inventory, reports, and more.

Latest highlights from What's New:

- Context-aware AI for ads, posters, banners, and social content
- AI-assisted commission and discount setup
- Vendor and seller price fields with live cart reflection
- Pincode-based availability checks and cart enforcement
- Improved booking, product page, and checkout experiences

See what's new in the project wiki: [What's New](https://github.com/BlupGithub/1D2C.com/wiki/What%27s-New).
